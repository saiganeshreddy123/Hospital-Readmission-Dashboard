# Hospital 30-Day Readmission Dashboard (Power BI)

Interactive dashboard built with de-identified U.S. EHR data to help hospitals reduce preventable readmissions (CMS estimates $26B annual cost).

## Key Features
- 30-day readmission rates by diagnosis (Heart Failure, COPD, Pneumonia, etc.)
- Risk stratification and drill-down by payer, age, and length of stay
- Comparison to national benchmarks
